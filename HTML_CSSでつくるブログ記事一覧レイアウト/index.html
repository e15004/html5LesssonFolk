Keep coding everyday (2013.01.01)

コードを書く喜びとして、一つの問題を解決した時の達成感が挙げられます。

ある問題が発生した時に、頭を悩ませながら色々なことを試して答えに近づいていく過程はプログラマーにとって至福の一時と言えます。

そうして沢山の問題を解決しているうちにだんだん問題のパターンが見えてきて… 続きを読む
